### 1.2.1 基本概念
##### **定理**： #伯努利方程
> <font color="#ccc1d9">描述：</font> 在线性微分方程的基础上，在 $Q(x)$ 的右边乘上 $y^{1-\alpha}=u)$：
> $$y^{\prime}+p (x) y=Q (x) y^{\alpha} \quad\quad\quad\quad\quad\quad(\alpha\neq1)\quad(y^{1-\alpha}=u)$$ 

**解释**
+ 求解：
	+ 将 $y$ 设为 $y^{1-\alpha}=u$，使得方程变成一阶微分线性方程；
+ 方法：
	+ 如果 $\alpha$ 等于 0，那就是线性方程；
	+ 如果 $\alpha$ 等于 1，那就直接可以使用可分离变量；
	+ 因此：当前的讨论，都是假设阿法及不等于 0，也不等于 1；
+ 思路：
	+ 思考如何才能化成线性方程；

**处理方法**
+ 已知：$y^{\prime}+p (x) y=Q (x) y^\alpha$
+ 第一步
	+ 把 $y^\alpha$ 除到等式的左边来，和 y 相除，得到：
	+ $y^{1-\alpha}$
+ 第二步：设 z
	+ 令 $y^{1-\alpha}=z$
+ 第三步
	+ 化成线性方程

### 1.2.2 例题
**例题**：$\frac{dy}{dx}+\frac yx=a(\ln x)y^2$
### 一、首先：明确“伯努利方程”的定义
伯努利方程的**标准形式**为：  
$$\frac{dy}{dx} + P(x)y = Q(x)y^n$$  
其中：  
- $P(x)$、$Q(x)$ 是关于 $x$ 的已知连续函数；  
- $n$ 是常数，且 $n \neq 0,1$（若 $n=0$ 则为一阶线性非齐次方程，$n=1$ 则为可分离变量方程，均不属于伯努利方程）。  
对比题目中的方程 $\frac{dy}{dx} + \frac{y}{x} = a(\ln x)y^2$：  
- $P(x) = \frac{1}{x}$（仅含 $x$ 的函数）；  
- $Q(x) = a\ln x$（仅含 $x$ 的函数）；  
- $n=2$（$y$ 的次数为 2，且 $2 \neq 0,1$）。  
完全满足伯努利方程的定义，因此题型判定正确。
### 二、其次：例题解法的逻辑分析（为何这样代换？）
伯努利方程的核心解法是**通过变量代换，将其转化为“一阶线性非齐次微分方程”**（因为一阶线性方程有成熟的通解公式，可直接套用）。  
代换的关键思路：针对 $y^n$ 项，令 $z = y^{1-n}$（消去 $y$ 的高次项）。  
本题中 $n=2$，因此令 $z = y^{1-2} = y^{-1}$（即 $\frac{1}{y}$），这正是例题中“设 $y$ 的负一次方为 $z$”的原因。  
### 三、完整解析步骤

#### 步骤1：将方程化为“便于代换”的形式
原方程：$\frac{dy}{dx} + \frac{y}{x} = a(\ln x)y^2$  
两边同时除以 $y^2$（$y \neq 0$，后续需验证 $y=0$ 是否为解），消去右边的 $y^2$：  
$$\frac{1}{y^2}\frac{dy}{dx} + \frac{1}{x} \cdot \frac{1}{y} = a\ln x \tag{1}$$
#### 步骤2：变量代换，转化为一阶线性方程
令 $z = \frac{1}{y}$（即 $z = y^{-1}$），对 $x$ 求导（利用链式法则）：  
$$\frac{dz}{dx} = \frac{d}{dx}(y^{-1}) = -1 \cdot y^{-2} \cdot \frac{dy}{dx} = -\frac{1}{y^2}\frac{dy}{dx}$$  
变形得：$\frac{1}{y^2}\frac{dy}{dx} = -\frac{dz}{dx}$  

将 $\frac{1}{y^2}\frac{dy}{dx} = -\frac{dz}{dx}$ 和 $\frac{1}{y} = z$ 代入方程 (1)：  
$$-\frac{dz}{dx} + \frac{1}{x} \cdot z = a\ln x$$
整理为**一阶线性非齐次方程的标准形式**（$\frac{dz}{dx} + P(x)z = Q(x)$）：  
$$\frac{dz}{dx} - \frac{1}{x}z = -a\ln x \tag{2}$$
#### 步骤3：套用一阶线性方程的通解公式
一阶线性方程 $\frac{dz}{dx} + P(x)z = Q(x)$ 的通解公式为：  
$$z = e^{-\int P(x)dx} \left[ \int Q(x) \cdot e^{\int P(x)dx} dx + C \right]$$
针对方程 (2)，先计算核心积分：  
1. 求 $P(x) = -\frac{1}{x}$ 的积分：$\int P(x)dx = \int -\frac{1}{x}dx = -\ln|x| = \ln\frac{1}{|x|}$（不妨设 $x>0$，简化为 $\ln\frac{1}{x}$）；  
2. 计算积分因子 $e^{\int P(x)dx} = e^{\ln\frac{1}{x}} = \frac{1}{x}$；  
3. 计算 $e^{-\int P(x)dx} = e^{\ln x} = x$。  

将上述结果代入通解公式：  
$$z = x \cdot \left[ \int (-a\ln x) \cdot \frac{1}{x} dx + C \right]$$

#### 步骤4：计算积分，还原变量
先计算积分 $\int (-a\ln x) \cdot \frac{1}{x} dx$（令 $u = \ln x$，则 $du = \frac{1}{x}dx$）：  
$$\int (-a\ln x) \cdot \frac{1}{x} dx = -a \int u du = -a \cdot \frac{u^2}{2} + C_1 = -\frac{a}{2}(\ln x)^2 + C_1$$

因此：  
$$z = x \cdot \left[ -\frac{a}{2}(\ln x)^2 + C \right] = -\frac{a}{2}x(\ln x)^2 + Cx$$  

最后还原 $z = \frac{1}{y}$，得到原方程的通解：  
$$\frac{1}{y} = Cx - \frac{a}{2}x(\ln x)^2$$  
（或整理为 $y = \frac{1}{Cx - \frac{a}{2}x(\ln x)^2}$，其中 $C$ 为任意常数）  

此外，需验证 $y=0$：将 $y=0$ 代入原方程，左边 $\frac{dy}{dx} + \frac{y}{x} = 0$，右边 $a(\ln x) \cdot 0 = 0$，等式成立，因此 $y=0$ 也是原方程的解（可视为通解中 $C \to \infty$ 的极限情况）。
