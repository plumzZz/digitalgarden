##
##### **定义**： #齐次型方程
> <font color="#ccc1d9">描述：</font>简化后的方程中**所有非零项的指数相等**：$$\frac{dy}{dx}=\varphi(\frac{y}{x})$$
 
**解释**
+ 概念：
	+ 右端是关于 xy 的二元函数，但可以写成 $\frac{y}{x}$ 的二元函数；
+ 齐次：
	+ 右端作为一个二元函数，是一个**零次、齐次函数**；
	+ 即：dy 比 dx 是函数，且**关于 x，y 次数相等**


**方法**：三板斧
+ 第零步：整理成齐次方程的形式，左边只有 $\frac{dy}{dx}$，即 $\frac{dy}{dx}=\varphi(\frac{y}{x})$；
+ 第一步：**换元**：写下 $u=\frac{y}{x}$
+ 第二步：因为 $y=xu$，所以对 y 求导后，可以得到 $\frac{dy}{dx}=u+x\frac{du}{dx}$ ，即得到了 $y^{\prime}=\frac{dy}{dx}=u+x\frac{du}{dx}=u+xu^{\prime}$ 的表达式；
+ 第三步：带会原函数，得到 $u+x\frac{du}{dx}=\varphi(\frac{y}{x})$，这个方程是可分离变量；
	+ 即：把其变成可分离变量，求解；


**例题**：$\text{求微分方程 }x^2y^{\prime}+xy=y^2\text{ 满足初始条件}y(1)=1的特解$
+ 分析
该微分方程通过变形可转化为齐次方程的形式，再利用变量代换的方法将其转化为可分离变量的方程，最后结合给定的初始条件求出特解。
解析步骤 1（将方程化为齐次方程形式）：
原方程$x^{2}y^{\prime}+xy = y^{2}$，可变形为$y^{\prime}=\frac{y^{2}-xy}{x^{2}}=\left(\frac{y}{x}\right)^{2}-\frac{y}{x}$。令$u = \frac{y}{x}$，即$y = ux$，那么$y^{\prime}=u^{\prime}x + u$。

步骤 2（代入并分离变量）：
将$y = ux$和$y^{\prime}=u^{\prime}x + u$代入$y^{\prime}=\left(\frac{y}{x}\right)^{2}-\frac{y}{x}$，得到$u^{\prime}x + u = u^{2}-u$，整理得$u^{\prime}x = u^{2}-2u$，进一步分离变量为$\frac{du}{u^{2}-2u}=\frac{dx}{x}$。

步骤 3（积分求解）：
对$\frac{du}{u^{2}-2u}=\frac{dx}{x}$两边积分，先对左边部分分式分解：$\frac{1}{u^{2}-2u}=\frac{1}{u(u - 2)}=\frac{1}{2}\left(\frac{1}{u - 2}-\frac{1}{u}\right)$。
所以积分得$\frac{1}{2}\int\left(\frac{1}{u - 2}-\frac{1}{u}\right)du=\int\frac{dx}{x}$，即$\frac{1}{2}\left(\ln|u - 2|-\ln|u|\right)=\ln|x|+C_{1}$（$C_{1}$为常数）。
化简得$\ln\left|\frac{u - 2}{u}\right|=2\ln|x|+C_{2}$（令$C_{2} = 2C_{1}$），进一步$\frac{u - 2}{u}=Cx^{2}$（$C = \pm e^{C_{2}}$）。

步骤 4（回代并利用初始条件）：
将$u = \frac{y}{x}$回代，得到$\frac{\frac{y}{x}-2}{\frac{y}{x}}=Cx^{2}$，化简为$\frac{y - 2x}{y}=Cx^{2}$。
利用初始条件$y(1) = 1$，代入得$\frac{1 - 2\times1}{1}=C\times1^{2}$，即$C = - 1$。
所以方程变为$\frac{y - 2x}{y}=-x^{2}$，整理得$y - 2x=-x^{2}y$，移项得$y + x^{2}y=2x$，即$y(1 + x^{2})=2x$，所以$y=\frac{2x}{1 + x^{2}}$。


**例题**：$y^{2}+x^{2}\frac{dy}{dx}=xy\frac{dy}{dx}$
+ 分析
	+ 注意：
	+ 1. 先对函数处理，处理成函数满足 y/x 的形式；
	+ 2. 如何判断是否是齐次方程 -> x 和 y 的次数比较对称；
+ 解析步骤 1（原方程）：$y^{2}+x^{2}\frac{dy}{dx}=xy\frac{dy}{dx}$

步骤 2（移项整理）：将含$\frac{dy}{dx}$的项移到一侧，得$y^{2} = xy\frac{dy}{dx}-x^{2}\frac{dy}{dx}$，提取公因式$\frac{dy}{dx}$，即$y^{2} = \frac{dy}{dx}(xy - x^{2})$

步骤 3（分离变量）：变形为$\frac{dy}{dx} = \frac{y^{2}}{xy - x^{2}} = \frac{y^{2}}{x(y - x)}$，分子分母同除以$x^{2}$，得$\frac{dy}{dx} = \frac{(\frac{y}{x})^{2}}{\frac{y}{x} - 1}$

步骤 4（换元简化）：设$u = \frac{y}{x}$，则$y = ux$，$\frac{dy}{dx} = u + x\frac{du}{dx}$，代入方程得$u + x\frac{du}{dx} = \frac{u^{2}}{u - 1}$

步骤 5（分离变量并积分）：移项得$x\frac{du}{dx} = \frac{u^{2}}{u - 1}-u = \frac{u^{2}-u(u - 1)}{u - 1} = \frac{u}{u - 1}$，分离变量为$\frac{u - 1}{u}du = \frac{1}{x}dx$，两边积分$\int(1 - \frac{1}{u})du = \int\frac{1}{x}dx$

步骤 6（计算积分结果）：积分得$u - \ln|u| = \ln|x| + C$（$C$为常数）

步骤 7（回代$u$并整理）：将$u = \frac{y}{x}$代入，得$\frac{y}{x} - \ln|\frac{y}{x}| = \ln|x| + C$，化简为$\frac{y}{x} = \ln|\frac{y}{x}| + \ln|x| + C = \ln|y| + C$

最终得出通解：$\boxed{y = x\ln|y| + Cx}$（$C$为常数）
