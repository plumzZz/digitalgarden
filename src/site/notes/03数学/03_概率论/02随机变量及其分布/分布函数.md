# 分布函数
分布函数全称为**累积分布函数（Cumulative Distribution Function，简称 CDF）**，用于描述随机变量 X 取值 “小于等于某个固定值 x” 的概率。
- **输入与输出**：输入是任意实数 x，输出是一个 “概率值”（范围在 [0,1] 之间）。
- **累积性**：F(x) 描述的是 X 取所有 “小于等于 x” 的可能值的概率总和（“累积” 的含义）。
- **通用性**：无论 X 是离散型（如掷骰子的点数）还是连续型（如灯泡的寿命），分布函数都适用，是统一描述两类随机变量的工具。

### 二、分布函数的核心性质（判定依据）

一个函数 F(x) 能成为某随机变量的分布函数，必须满足以下 4 条基本性质（缺一不可）：
1. **非负性与有界性**：对任意 x，0≤F(x)≤1
    - 最小值：limx→−∞​F(x)=0（当 x 趋近于负无穷时，X≤x 是不可能事件，概率为 0）。
    - 最大值：limx→+∞​F(x)=1（当 x 趋近于正无穷时，X≤x 是必然事件，概率为 1）。
2. **单调不减性**：若 x1​<x2​，则 F(x1​)≤F(x2​)
    - 含义：当 x 增大时，“X≤x” 的范围扩大，累积概率不会减少。
3. **右连续性**：对任意 x0​，limx→x0+​​F(x)=F(x0​)
    - 含义：当 x 从 x0​ 的右侧（即比 x0​ 大的方向）趋近于 x0​ 时，函数值连续；左侧可能存在跳跃（离散型随机变量的典型特征）。

# 一、概率密度的严格定义
对于连续型随机变量 X（比如 “人的身高”“灯泡寿命”“公交车等待时间” 等取值能充满某一区间的变量），如果存在一个非负可积函数 $f(x)$，使得对任意实数 $a \leq b$，随机变量 X 落在区间 $[a, b]$ 内的概率可以表示为：$P(a \leq X \leq b) = \int_{a}^{b} f(x) dx$
那么这个函数 $f(x)$ 就称为随机变量 X 的概率密度函数（简称 “概率密度”）。
