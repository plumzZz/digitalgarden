要理解 $P\{X \leq 1, |X| \leq 1\} = P\{|X| \leq 1\}$，需从**事件的包含关系**和**概率的基本性质**分析：


### 步骤1：分析事件的“范围包含”
事件 $\{X \leq 1, |X| \leq 1\}$ 表示“$X \leq 1$ 且 $|X| \leq 1$”；
事件 $\{|X| \leq 1\}$ 表示“$-1 \leq X \leq 1$”。

现在看两个事件的范围：
- 若 $|X| \leq 1$（即 $-1 \leq X \leq 1$），则必然满足 $X \leq 1$（因为 $1$ 是上限）。
- 因此，“$|X| \leq 1$”这个事件发生时，“$X \leq 1$ 且 $|X| \leq 1$”必然发生，即：
  
$$
\{|X| \leq 1\} \subseteq \{X \leq 1, |X| \leq 1\}
$$



### 步骤2：再看反向的包含关系
反过来，事件 $\{X \leq 1, |X| \leq 1\}$ 要求“$X \leq 1$ 且 $|X| \leq 1$”，而“$|X| \leq 1$”本身就隐含“$X \leq 1$”（因为 $|X| \leq 1$ 意味着 $X \leq 1$）。
因此，“$X \leq 1$ 且 $|X| \leq 1$”等价于“$|X| \leq 1$”，即：
  
$$
\{X \leq 1, |X| \leq 1\} \subseteq \{|X| \leq 1\}
$$



### 步骤3：由“互相包含”得“事件相等”
因为 $\{|X| \leq 1\} \subseteq \{X \leq 1, |X| \leq 1\}$ 且 $\{X \leq 1, |X| \leq 1\} \subseteq \{|X| \leq 1\}$，根据“集合相等的定义（互相包含则相等）”，可得：
  
$$
\{X \leq 1, |X| \leq 1\} = \{|X| \leq 1\}
$$



### 步骤4：由“事件相等”得“概率相等”
根据概率的**基本性质**：“若两个事件 $A = B$，则 $P(A) = P(B)$”。
由于 $\{X \leq 1, |X| \leq 1\} = \{|X| \leq 1\}$，因此：
  
$$
P\{X \leq 1, |X| \leq 1\} = P\{|X| \leq 1\}
$$



### 总结
核心逻辑是：**“$|X| \leq 1$”已经包含了“$X \leq 1$”的要求**，因此“$X \leq 1$ 且 $|X| \leq 1$”和“$|X| \leq 1$”是同一个事件，概率自然相等。
