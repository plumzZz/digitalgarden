## 22.1 惯性定理 
**概念**：惯性定理
+ 无论选取什么样的可逆线性变换，将$二次型化成标准形或规范形$，其正项个数 `p`，负项个数 `q` 都是不变的；
+ 其中 `p` 称之为正惯性指数；
+ 其中 `q` 称之为负惯性指数；
+ 且：$若二次型的秩为r，则r=p+g，可逆线性变换不改变正、负惯性指数$

**概念**：两个二次型 (或实对称矩阵)$合同的充要条件$ 
+ 概念： 
	+ 两个二次型 (或实对称矩阵)合同的充要条件是$有相同的正、负惯性指数, 或有相同的秩及正 (或负)惯性指数，或有相同的正、负特征值个数$
+ 正交矩阵下：
	+  两个矩阵有相同的正、负特征值的个数；
