8. 一个等式右边是特征值矩阵，等式左边对应的解向量矩阵乘上一个矩阵，那么这个特征值应该会怎么变化？
	1. $ 三、推导：相似变换保持特征值不变$
	2. $由 (AP = P\Lambda)，可得 (A = P\Lambda P^{-1})（因 P 可逆，可对角化）。$
	3. $将 (A = P\Lambda P^{-1}) 代入等式 (A(PB) = (PB)Lambda')，左边展开：(A(PB) = (P\Lambda P^{-1})(PB) = P\Lambda B)$
	4. $右边为 ((PB)\Lambda')，因此等式变为：(P\Lambda B = PB\Lambda')$
	5. $由于 P 可逆（特征向量矩阵线性无关），两边左乘 (P^{-1})，得：(\Lambda B = B\Lambda')$
	6. $整理得：(\Lambda' = B^{-1}\Lambda B)$
	### $四、结论：特征值的 “不变性” 与 “相似性”$
	$由 (\Lambda' = B^{-1}\Lambda B) 可知：$
	- $(\Lambda') 与 (\Lambda) 是相似矩阵（相似变换矩阵为 B）。$
	- $相似矩阵的特征值完全相同（包括重数，因特征多项式 (\det(\lambda I - \Lambda') = \det(\lambda I - \Lambda))）。$
